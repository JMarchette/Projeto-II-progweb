Conteúdo extra:

Além do uso de CSS e HTML, é possível utilizar linguagens como JavaScript e python para desenvolver o back-end de um site.
JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, usada principalmente para adicionar interatividade aos sites e aplicações web. É uma das linguagens de programação mais populares do mundo, e é usada por milhões de desenvolvedores em todo o mundo.
Com JavaScript é possível:
  • Adicionar animações, formulários dinâmicos e jogos


  • Manipular dados, como carregar dados do servidor, armazenar dados no cliente e atualizar dados em tempo real


  • Implementar autenticação e autorização, como login de usuários e controle de acesso


  • Validar dados, como verificar se os dados inseridos pelo usuário são válidos


  • Testar sites e aplicações web
